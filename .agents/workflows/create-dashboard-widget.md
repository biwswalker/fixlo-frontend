# Create Dashboard Widget Workflow

1. Ask the user which specific KPI or data point needs to be displayed on the widget.
2. Read `@.agents/rules/flexio-core.md` to check the relevant PostgreSQL database table schema.
3. Write a Next.js Server Action to fetch and aggregate the requested data.
4. Create a responsive UI component using Recharts or a shadcn/ui Card.
5. Provide a clear summary of the implemented changes.
